Pasos para utilizar el programa:
1-Inicializar 2 instancias de Visual Studio
2-Abrir el programa, puede ser abriendo el Form1.cs
3-En IP se recomienda colocar en ambos 127.0.0.1
4-En puerto se puede usar cualquier en el rango 1024-6365535 en ambos programas debe ser el mismo.
5-En uno de los programas darle escuchar para habilitarse como Servidor
6-En el otro programa presionar conectar para conectarse como cliente al servidor.
7-Escribir el mensaje deseado a enviar y darle a enviar.
8-En el otro programa se vera el mensaje recibido
